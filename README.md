# UK Cost of Living: 1965 vs 1995 vs 2025

A comparative analysis of the cost of living in the UK across three generations, visualizing the purchasing power paradox.

## Overview

This is a static web application that visualizes economic data to compare affordability of housing, education, and daily expenses relative to income.

## Features

- **Interactive Dashboard**: Switch between Single Earner and Dual Income Couple profiles.
- **Data Visualization**: Chart.js powered graphs showing income vs house prices, wallet share, and big ticket items.
- **Responsive Design**: Built with Tailwind CSS for mobile and desktop support.

## Project Structure

- `index.html`: Main entry point and layout.
- `styles.css`: Custom styles and overrides.
- `script.js`: Application logic and chart configurations.

## Deployment on Vercel

This project is optimized for deployment on Vercel.

1.  Push this repository to GitHub/GitLab/Bitbucket.
2.  Import the project in Vercel.
3.  Deploy (no build configuration required).

Alternatively, drag and drop the folder into the Vercel CLI.
